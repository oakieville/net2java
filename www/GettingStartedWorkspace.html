Welcome to the .NET2Java Workspace

Danny Coward January 2007
As soon as you check all the files you need to do these THREE things.

cd to the root of the workspace you checked out

1) Build the Library

cd library
ant build

2) Build the parser

cd dnj
ant build
ant openprojects
ant vbtests
ant cstests
ant demos
(all the tests and demos should work otherwise something has gone wrong)

3) Build the NetBeans Plugin

cd dnj
ant plugin  (copies the parser source into the plugin project...)
start NetBeans 5.5
Open dnj/plugin/DNPlugin
Build the plugin from the NetBeans Menu

Other things to know

If you add a test, remember to do

cd dnj
ant closeprojects

in order to update the projects archive.

To use Net2Java on the command line...

cd dnj

ant cmd     for command line apps
ant gui     for web apps
ant web     for web apps

The parser picks up the project details to translate from the dnj/project.properties file. Its pretty self evident.

And finally...

Take a tour of build.properties to see where the command line version picks up the library and other settings. And check out build.xml, there may be some other targets of interest in there.

Have fun !
